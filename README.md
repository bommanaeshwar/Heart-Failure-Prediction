# Heart Failure Prediction using Machine Learning

## Overview
This project aims to predict the likelihood of heart disease using Machine Learning and Deep Learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and model evaluation to identify patterns associated with heart disease.

## Objectives
- Analyze the heart disease dataset.
- Perform data preprocessing and feature engineering.
- Train Machine Learning and Deep Learning models.
- Evaluate model performance using classification metrics.
- Compare different approaches for heart disease prediction.

## Dataset
The dataset contains patient health information such as:
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

**Target Variable:**
- HeartDisease
  - 0 = No Heart Disease
  - 1 = Heart Disease
